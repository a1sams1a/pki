# SPARCS PKI
SPARCS Public Key Infrastructure

## Certificate Chain
```
+ CN=SPARCS (RSA 4096, sha512RSA)
+-- CN=<sparcs-id> (RSA 4096, sha512RSA) - for users
+-- CN=<domain-name> (RSA 4096, sha512RSA) - for services
```

## CRL
* http://cert.sparcs.org/sparcs.crl (max: 3days, renew: 1day)
